Coding Challenge:
Metaballs Refactoring

Coding challenge #28 metaballs from the Coding Train refactored to be ojbect oriented. This refactoring was to make the current metaballs 'teleport' to another location, this was done by adding a function called mouseIsPressed. I used this function because clicking made my refactoring easier as I could click anywhere on the canvas to 'teleport' the metaballs.
Link: https://editor.p5js.org/kevin-dang4/sketches/Qfuvz6M6d

Coding Challenge:
Snake Creative Refactoring

Coding challenge #3 snake game from the Coding Train refactored to be creative. I just made another file called anotherSnake and made another snake so there would be two instead of one. Instead of making it controlled by another set of keys both of the snakes are controlled by one person.
Link: https://editor.p5js.org/kevin-dang4/sketches/ExIny4Ow9
